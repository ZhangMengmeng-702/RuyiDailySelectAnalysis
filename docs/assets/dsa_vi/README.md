## 设计理念
Logo 以三根 K 线柱为主体，构成"如意"的几何抽象，顶部金线与如意云纹曲线呼应：
- **绿柱** → 多头趋势、AI 量化
- **金柱** → 如意吉祥、金融属性
- **紫柱** → 科技感、智能化

渐变由翡翠绿 -> 琥珀金 -> 科技紫，16px 仍清晰可辨。
